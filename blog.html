Question 1: What are the different ways to select an element in the DOM?

Answer: There are several ways to select elements in the DOM:

  • getElementById() - Selects a single element by its ID.
  • getElementsByClassName() - Selects all elements with a specific class.
  • getElementsByTagName() - Selects all elements of a given tag name.
  • querySelector() - Selects the first element that matches a CSS selector.
  • querySelectorAll() - Selects all elements that match a CSS selector.

Question 2: What is the difference between innerHTML, innerText, and textContent?

Answer:

  • innerHTML - Gets or sets the HTML content of an element, including tags.
  • innerText - Gets or sets the visible text of an element, excluding hidden elements.
  • textContent - Gets or sets all the text within an element, including hidden elements, but ignores HTML tags.

Question 3: What is event delegation in the DOM?

Answer: Event delegation is a technique where a single event listener is added to a parent element to manage events for its children. It leverages event bubbling to efficiently handle events for multiple child elements.

Question 4: What is event bubbling in the DOM?

Answer: Event bubbling is a mechanism in the DOM where an event triggered on a child element first runs its handler and then propagates up to its ancestors, executing their respective event handlers if they exist.

Question 5: How do you create, add, and remove elements using JavaScript?

Answer:

  • Create: Use document.createElement() to create a new element.
  • Add: Use appendChild() or append() to add the element to the DOM.
  • Remove: Use removeChild() or remove() to delete an element from the DOM.
